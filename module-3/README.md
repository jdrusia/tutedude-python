# Module 3 Control Structures in Python

This program has two exercises:

- Exercise 1 checks if a number entered by the user is even or odd.
- Exercise 2 calculates the sum of numbers from 1 to 50 using a for loop.

How to run:

1. Run the command: python combined_tasks.py.
2. Then choose option 1 or 2 from the menu.

Example output:
